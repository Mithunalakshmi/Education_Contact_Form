# 🎓 EduLearn Contact Form

A simple education-themed contact form built using **HTML, CSS, JavaScript, Node.js, Express.js, and Nodemailer**.

## 🚀 Features

* Responsive contact form
* Sends form data to backend
* Automatic email response using Nodemailer
* Error handling and CORS support

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* Nodemailer

## ▶️ Run the Project

Install dependencies:

```bash
npm install
```

Start the server:

```bash
node server.js
```

Server runs on:

```text
http://localhost:5000
```
